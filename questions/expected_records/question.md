# Expected Number of Records

**Type:** Probability / Combinatorics
**Tags:** Expectation, Linearity of expectation, Permutations, Harmonic numbers
**Date:** 2026-03-11

---

You observe a sequence that is a uniformly random permutation of $1, 2, \ldots, n$.

Element $\sigma(k)$ is called a **record** (or left-to-right maximum) if $\sigma(k) > \sigma(j)$ for all $j < k$.

**What is the expected number of records?**
